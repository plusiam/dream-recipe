# 🍳 나다움 레시피

> '나'라는 요리를 짓는 한 장의 레시피 카드 — 되어가고 싶은 나와, 그 길에 필요한 미덕을 학생이 직접 설계합니다.

**👉 직접 체험: https://plusiam.github.io/dream-recipe/**

---

## 앱 목적

학생이 *되어가고 싶은 나*(인재상)와 그것을 기르는 데 *필요한 재료*(미덕)를 스스로 골라, 자기 이해와 성찰을 담은 **디지털 '나다움 선언'** 카드를 만듭니다. 단계를 진행할 때마다 앞서 고른 것이 누적되어 보여, 그것을 바탕으로 다음을 성찰하도록 돕습니다.

## 학년 · 교과

- **6학년 도덕** 1단원 「나는 꿈 요리사」
- **성취기준**: [6도01-02] 반성·성찰(자기 이해)
- 개념 렌즈: 정체성 · 나다움 · 미덕 · 성장

## 사용 시나리오

1. 학생이 태블릿·크롬북으로 접속합니다.
2. 되어가고 싶은 나(IB 학습자상)를 고릅니다.
3. 그 요리에 필요한 재료(미덕)를 2~3개 고르거나 직접 더합니다.
4. 지금 내 부엌에 이미 있는 재료(받아들이기)를 적습니다.
5. 오늘 해볼 작은 한 가지(한 꼬집)를 적습니다.
6. 완성된 레시피 카드를 **PNG로 저장 · 인쇄 · 공유**합니다.

> 공유는 선택이며, 카드 저장·인쇄는 누구나 할 수 있습니다. 비어 있는 칸이 있어도 괜찮은 '빈 곳의 원칙'을 따릅니다.

## 설계 원칙 (교육적 가드레일)

- **무순위·무점수** — 미덕에 등급이나 점수가 없습니다.
- **자기수용을 출발점으로** — '지금 내 부엌엔'은 부족이 아니라 이미 가진 재료입니다.
- **인재상·미덕 강요 금지** — 학생이 자기 요리에 맞게 재료를 자유롭게 고릅니다.
- **빈 곳의 원칙** — 못 채운 칸도 괜찮습니다. 레시피는 천천히 완성됩니다.

## 기술

- 단일 `index.html` (HTML/CSS/JS 인라인, 프레임워크 없음)
- 폰트 Pretendard, 카드 저장 html2canvas (CDN)
- 백엔드 없음 · 완전 클라이언트 사이드
- 큰 글자 · 큰 터치 영역 · 모바일/태블릿 반응형 · 인쇄(A4) 지원
